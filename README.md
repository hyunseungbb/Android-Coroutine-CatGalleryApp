## Sequence

1. 서버에 XML 요청.
2. XML문서로부터 Image url을 파싱한다.
3. 파싱한 데이터를 RecyclerVIew에 추가 업데이트한다.
4. RecyclerView의 스크롤을 끝까지 내리면 위의 내용을 반복한다.
